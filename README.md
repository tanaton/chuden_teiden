# chuden_teiden
[中部電力の停電情報](http://teiden.chuden.jp/p/sizuoka.html)を町名ごとにまとめて表示します。  
遠州地方のみに対応しています。  
個人の私的利用の範囲でお使いください。

## 使い方
1. `git clone https://github.com/tanaton/chuden_teiden.git`
1. `cd chuden_teiden`
1. `go build chuden_teiden.go`
1. `./chuden_teiden`
1. ブラウザで[http://127.0.0.1/](http://127.0.0.1/)にアクセスする
